Overview - Employee Portal Time Off, Leave Management

Makes it easy for employee in your organization to manage time off from Portal.

Key Features

Compatible with Community and Enterprise

Manage Time Off request from portal.

Remaining / Taken Time Off summary dashboard.

Fully configured with Time Off type configuration.

From website portal, portal user can manage Time off with all core features.

Editable Time Off request as per stages.

Can chat with Time Off Manager.

Configuration

Set portal access to your user.

User is linked with employee.

Grant Portal access if user have not already signed up.

Set In Portal True, Send Email to invite signup process.

Details

Portal user can manage Time Off request from My Account.

Create New Time Off request from Create New button.

Fill the details of Form and Confirm.

Edit Time Off request.

Time Off Manager can see requested Time Off and can Approve or Refuse.

Portal User can see status, No of days, start and end date, description and Type of requested Time Off.

It allows to select only Allocated day(s) and hour(s).

Half day and Custom hours selection is based on Time Off configuration.

Validations

Required field validation.

Overlap day(s) validation.

Time Off validity validation.

Remaining Time Off in-sufficient validation.

Filters

Sort By, Filter By, Group By.

Time Off Summary - Dashboard

Time Off summary calculated on approved time off requests.
